将你的创意随心所欲地书写在屏幕的任意角落。  
智绘教Inkeys 拥有丝滑流畅的高性能画笔、丰富强大的功能，以及众多贴心小设计，全面提升你的效率与使用体验。  
软件基于 C++20 编写，专为 Windows 平台打造。  

原名 `Intelligent-Drawing-Teaching`（简称 IDT）
